3.2 三分钟快速部署（以Ubuntu为例）

```bash

1. 升级系统

sudo apt update && sudo apt upgrade -y

2. 安装Shadowsocks（轻量级代理）

sudo apt install shadowsocks-libev -y

3. 配置加密通道

sudo nano /etc/shadowsocks-libev/config.json
配置文件示例：json
{
"server":"0.0.0.0",
"serverport":8388,
"password":"YourSecurePassword",
"method":"aes-256-gcm",
"fastopen":true
}
```
